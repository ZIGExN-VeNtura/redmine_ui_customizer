# Redmine UI Customizer

A Redmine plugin to customize UI elements across pages.

## Features

| Feature | Page | Description |
|---------|------|-------------|
| Hide delete button | `/users` | Hide delete button in context menu |

## Installation

1. Copy plugin folder to `plugins/redmine_ui_customizer`
2. Restart Redmine

## Configuration

Go to **Administration → Plugins → Redmine UI Customizer → Configure**

Enable/disable features as needed.

## Compatibility

- Redmine 6.x
